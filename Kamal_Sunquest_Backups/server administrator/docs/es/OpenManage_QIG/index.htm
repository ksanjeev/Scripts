Maximización del valor del sistema Dell™ PowerEdge™

Use los CD proporcionados para instalar y poner en marcha rápidamente el sistema. Obtendrá resultados confiables y consistentes y asegurará una operación sin problemas. Los CD proporcionan:

  • Instalación mejorada del sistema operativo: reduce el tiempo requerido para la instalación de los sistemas operativos Microsoft® Windows®, Red Hat® Enterprise Linux y SUSE® LINUX Enterprise Server, gracias a que guía al usuario a través de un proceso paso a paso y fácil de seguir.

  • Instalación y configuración: proporciona las herramientas necesarias para instalar y configurar el software y los sistemas PowerEdge. Las herramientas permiten la configuración y descubrimiento automáticos de los adaptadores de red y controladores RAID proporcionados por Dell.

  • Utilidades, archivos controladores y diagnósticos: permiten ver y crear discos tanto de diagnóstico como de controladores de software de Dell.

  • Systems Management: instala las aplicaciones de software Dell OpenManage™ para asegurar máxima confiabilidad y disponibilidad de los sistemas. Estas aplicaciones proporcionan las herramientas necesarias para supervisar, diagnosticar y actualizar los sistemas de manera remota.

  • Documentación: permite acceso rápido a los documentos acerca del sistema, systems management software, controlador RAID y de periféricos en los idiomas admitidos.

NOTA: Para ver la documentación acerca de sistemas que ejecutan los sistemas operativos Red Hat Enterprise Linux o SUSE LINUX Enterprise Server, debe abrir index.htm desde el directorio del idioma apropiado.

Comprobación del contenido del paquete de Dell OpenManage Systems Management Software

El paquete de systems management software incluye, entre otros, los siguientes componentes:

  • El CD Dell PowerEdge Installation and Server Management es un CD de inicio que proporciona las herramientas necesarias para configurar el sistema e instalar el sistema operativo. Este CD contiene los últimos productos del systems management software, incluso los servicios de acceso remoto, Storage Management y los diagnósticos de Dell OpenManage Server Administrator.

  • El CD Dell Systems Management Consoles contiene todos los productos más recientes de la consola de systems management, incluyendo Dell OpenManage IT Assistant.

  • El CD Dell PowerEdge Service and Diagnostic Utilities provee las herramientas necesarias para configurar el sistema, así como lo más reciente en diagnósticos y controladores para el sistema optimizados por Dell .

  • Los CD Dell PowerEdge Updates están incluidos sólo con el paquete de suscripción de servicio y ofrecen las versiones más recientes de software para el sistema a través de Dell OpenManage Server Update Utility (SUU) y Dell Update Packages (DUP). SUU se entrega en CD por separado para Microsoft Windows y Linux.

  • El CD Dell PowerEdge Documentation le ayuda a permanecer actualizado con la documentación para sistemas, productos del Systems Management Software, periféricos y controladores RAID.

La mayoría de estos CD además contienen archivos léame, que proveen la información más reciente sobre los productos.

Compare el contenido de la caja de accesorios del sistema con la lista de empaque o la factura que se incluyen con el sistema. Si algún componente está dañado o no está incluido, llame a Dell dentro de los 30 días siguientes a la fecha de factura, para obtener su reemplazo de manera gratuita. Para obtener más información, consulte "Obtención de asistencia técnica."

Introducción al Systems Management Software

Dell OpenManage Systems Management Software es un conjunto de programas de aplicación para sistemas PowerEdge y Dell™ PowerVault™. Este software permite administrar el sistema por medio de supervisión, diagnóstico, notificación y acceso remoto proactivos.

Cada sistema que planea administrar con software de Dell OpenManage recibe el nombre de sistema administrado. Las aplicaciones de Managed system incluyen el software Server Administrator y el software remote access controller (RAC).

La estación de administración se puede utilizar para administrar de manera remota, uno o más sistemas administrados desde una ubicación central.

NOTA: Dell OpenManage Array Manager Console (para management station) está disponible para Windows solamente si se detecta el software Dell OpenManage management station de versión previa (con consola Array Manager instalada). Solamente está disponible para actualización.

Instalación del sistema operativo utilizando Server Assistant

Realice los siguientes pasos para determinar si el sistema tiene sistema operativo instalado:

  1. Compruebe que el tablero, el mouse y el monitor estén conectados al sistema, y encienda el sistema.

  2. Lea y acepte el Contrato de licencia de software para continuar.

Si aparece un mensaje indicando que no existen unidades de disco de inicio o que no se encontró un sistema operativo, entonces el sistema no tiene sistema operativo. Tenga el CD del sistema operativo a la mano y lleve a cabo el procedimiento para instalar un sistema operativo.

Si el sistema cuenta con un sistema operativo preinstalado no es necesario que realice el procedimiento de instalación. Localice el documento titulado Instrucciones de instalación del sistema operativo, que se proporcionó con el sistema, y siga dicho procedimiento para la instalación.

Realice los siguientes pasos para instalar un sistema operativo en su sistema:

  1. Inserte el CD Dell PowerEdge Installation and Server Management y reinicie el sistema.

Las siguientes opciones estarán disponibles durante el inicio del sistema:

    • Server Assistant: inicia a través del CD.

    • Server Assistant con CD de verificación de medios: verifica si el CD se creó correctamente y valida el contenido.

    • Omitir inicio con CD - Inicio con disco duro: inicia desde el disco duro y verifica si hay un sistema operativo instalado en el sistema. Si no hay un sistema operativo instalado en el sistema, inicia mediante el CD.

NOTA: Si no selecciona una opción en 10 segundos, el sistema automáticamente inicia mediante el CD.
  1. Seleccione Configuración del servidor en la página principal de Server Assistant.

  2. Siga las instrucciones paso a paso para configurar el hardware e instalar el sistema operativo.

Para obtener información adicional acerca de cómo instalar RAID, consulte Para iniciar con RAID en el CD Documentation.

NOTA: Cuando se utiliza Server Assistant para instalar un sistema operativo admitido, Server Assistant automáticamente copia en el escritorio los archivos relevantes de instalación de Systems Management Software en el disco duro y coloca los iconos Instalar Server Administrator y Eliminar archivos de instalación de Server Administrator.
NOTA: Puede usar el icono Instalar Server Administrator para instalar Server Administrator sin el CD de instalación. En los sistemas que ejecutan Windows, si se hace clic en este icono se ejecutará la interfaz de instalación estándar. En sistemas que ejecutan Red Hat Enterprise Linux o SUSE LINUX Enterprise Server, al hacer clic en este icono se ejecuta la secuencia de comandos de la instalación personalizada de Server Administrator. Si no desea instalar Server Administrator, puede eliminar los archivos de instalación haciendo clic en el icono Eliminar archivos de instalación de Server Administrator. Después de confirmar que desea continuar, todos los archivos de Server Administrator, incluyendo los iconos se eliminan.
NOTA: Al instalar el sistema operativo Windows, algunas configuraciones de hardware con más de 4 GB de memoria física instalada requieren de pasos adicionales después de la instalación del sistema operativo para que puedan utilizar completamente toda la memoria instalada.     Instalación del Systems Management Software en un sistema administrado

    El programa de configuración proporciona las opciones tanto Configuración rápida como Configuración personalizada. La opción Configuración rápida (recomendada) instala automáticamente todos los componentes de software detectados necesarios para administrar el sistema. La opción Configuración personalizada permite seleccionar los componentes de software que desea instalar. El procedimiento descrito en este documento se basa en la opción Configuración rápida. Para obtener información acerca de la Configuración personalizada consulte la Guía del usuario de instalación y seguridad de Dell OpenManage.

    AVISO: Dell OpenManage Array Manager ya no se admite. Si está actualizando un sistema (instalado con Dell OpenManage versión 4.3 o posterior) con Array Manager instalado, Array Manager se eliminará durante el proceso de actualización. Puede utilizar en su lugar el Storage Management Service de Server Administrator.
    NOTA: Para la instalación personalizada y otros detalles, consulte la Guía del usuario de instalación y seguridad. Management station y managed system software pueden instalarse en el mismo o distintos directorios. Puede seleccionar el directorio para la instalación.
    NOTA: Consulte el archivo "léame" de instalación incluido en el CD Installation and Server Management, en donde encontrará una lista de los sistemas actualmente admitidos. El archivo léame de instalación, readme_ins.txt, está localizado en el directorio srvadmin/docs/readme dentro del CD
    NOTA: Si hay una versión 4.2 o anterior (>=3.0) del software Systems Management instalada en el sistema, no se podrá actualizar directamente a la versión 5.0. Para conservar la configuración, debe actualizar a la versión 4.3, y después puede actualizar a la versión 5.0 para obtener las actualizaciones de software más recientes. La versión 4.3 está disponible en el sitio web de asistencia Dell en support.dell.com.
    NOTA: Instale el agente SNMP en su sistema administrado por medio de su sistema operativo antes de instalar el managed system software.
    NOTA: Para evitar iniciar en Server Assistant, retire el CD Installation and Server Management si va a reiniciar el sistema.

    Instalación del Managed System Software en sistemas operativos Microsoft Windows admitidos

    1. Inicie sesión con privilegios de administrador en el sistema que ejecuta el sistema operativo Windows admitido, donde desea instalar los componentes del Managed System.

    2. Compruebe que el sistema tenga instalado un explorador admitido.

    3. Si tiene un controlador RAID instalado en el sistema y planea instalar la función Storage Management, asegúrese que también estén instalados los controladores de dispositivos de cada controlador RAID.

    En el CD Service and Diagnostic Utilities encontrará controladores de dispositivos. Puede comprobar el controlador de dispositivo en Windows haciendo clic con el botón derecho del mouse en Mi PC, seleccionando Administrar y luego haciendo clic en Administrador de dispositivos.

    1. Inserte el CD Installation and Server Management en la unidad de CD.

    El programa de configuración debe iniciar automáticamente. En caso contrario, haga clic en el botón Inicio, haga clic en Ejecutar y luego escriba x:\srvadmin\windows\setup.exe (donde x es la letra de la unidad de CD). El programa de configuración inicia la utilidad de verificador de prerrequisitos.

    1. Revise los resultados de las verificaciones de prerrequisitos realizadas y corrija las condiciones que aparezcan en la lista.

    2. Haga clic en el botón Instalar, Modificar, Reparar, o Eliminar Server Administrator.

    3. Haga clic en Siguiente en la ventana de Bienvenida.

    4. Acepte el contrato de licencia y haga clic en Siguiente.

    5. Seleccione Configuración rápida y haga clic en Siguiente.

    6. Haga clic en Instalar para continuar con el proceso de instalación.

    El programa de instalación automáticamente instala todo el Managed System Software para la configuración de hardware.

    1. Cuando termine la instalación, haga clic en Terminar y si el programa se lo pide, reinicie el sistema antes de usar el software.

    Instalación del software Managed System para sistemas operativos admitidos Red Hat Enterprise Linux o SUSE LINUX Enterprise Server

    1. Inicie sesión como "root" en el sistema que ejecuta un sistema operativo admitido Red Hat Enterprise Linux o SUSE LINUX Enterprise Server, donde desea instalar los componentes de Managed System.

      NOTA: El sistema operativo SUSE LINUX Enterprise Server es admitido sólo en sistemas PowerEdge 1800, 1850, 2800 y 2850.

    2. Inserte el CD Installation and Server Management en la unidad de CD.

    3. Si es necesario, monte el CD utilizando el comando mount /dev/cdrom /media/cdrom, o un comando similar (para Red Hat Enterprise Linux 4 y SUSE LINUX Enterprise Server 9), O bien, mount /dev/cdrom /mnt/cdrom, o un comando similar (para Red Hat Enterprise Linux 3).

    4. Ejecute la secuencia de comandos srvadmin-install.sh desde el directorio

    /srvadmin/linux/supportscripts como sigue:

    sh srvadmin-install.sh --express

    o

    sh srvadmin-install.sh -x

    La secuencia de comandos instalará el conjunto de programas típico para la configuración del sistema.

    NOTA: Puede registrar la salida de la instalación RPM agregando 2>&1 | tee –a /var/log/srvadmin.log a la ejecución de la secuencia de comandos de shell mencionada anteriormente. EL comando resultante sería sh srvadmin-install.sh 2>&1|tee –a /var/log/srvadmin.log
    1. Inicie los servicios con el comando de inicio sh srvadmin-services.sh.

    Instalación de Managed Station Software para sistemas operativos admitidos Red Hat Enterprise Linux o SUSE LINUX Enterprise Server

    La Utilidad de administración de BMC y los componentes del RAC del conjunto de programas de management station pueden utilizarse en una estación de administración que ejecuta el sistema operativo Red Hat Enterprise Linux o SUSE LINUX Enterprise Server.

    NOTA: El sistema operativo SUSE LINUX Enterprise Server es admitido sólo en sistemas PowerEdge 1800, 1850, 2800 y 2850.

    2. Si es necesario, monte el CD utilizando el comando mount /dev/cdrom /media/cdrom o un comando similar (para Red Hat Enterprise Linux 4 y SUSE LINUX Enterprise Server 9) O bien, mount /dev/cdrom /mnt/cdrom o un comando similar (para Red Hat Enterprise Linux 3).

    Actualización de archivos controladores y utilidades

    El CD Service and Diagnostic Utilities contiene controladores de dispositivos y utilidades que puede utilizar para actualizar el sistema. Para poder utilizarlos, debe copiar los controladores y las utilidades ya sea a un disco o a la unidad de disco duro.

    Para obtener las actualizaciones más recientes de los productos, como, archivos controladores y aplicaciones de Dell OpenManage, visite el sitio web de asistencia Dell en support.dell.com.

    NOTA: El CD Service and Diagnostic Utilities sólo se puede ejecutar en sistemas que ejecutan Windows. Para extraer archivos controladores para sistemas que ejecutan Red Hat Enterprise Linux o SUSE LINUX Enterprise Server, debe utilizar el CD Service and Diagnostic Utilities en un sistema que ejecute Windows, y luego copiar o compartir los archivos seleccionados con el sistema de destino final.
    1. Inserte el CD Service and Diagnostic Utilities en la unidad de CD de un sistema que ejecute Windows.

    El programa de configuración debe iniciar automáticamente. En caso contrario, haga clic en el botón Inicio, haga clic en Ejecutar y luego escriba x:\setup.exe (donde x es la letra de la unidad de CD).

    1. Seleccione el sistema, y en Seleccionar controladores/utilidades, seleccione el conjunto de archivos que desea actualizar y luego haga clic en Continuar.

    2. Desde la Página de utilidades y controladores, elija el archivo que tenga la utilidad o el controlador que desee actualizar, haciendo clic en el vínculo del nombre del dispositivo.

    El sistema le preguntará en dónde desea guardar el archivo. Los archivos se guardan en formato comprimido. Los archivos de firmware normalmente requieren discos flexibles para la extracción.

    1. Descomprima el archivo.

    Consulte la Guía de compatibilidad de Dell OpenManage Server Administrator para obtener información adicional sobre las versiones de controladores y de las aplicaciones de Dell OpenManage.

    Los CD Dell PowerEdge Updates están incluidos sólo con el paquete de suscripción de servicio y ofrece las versiones más recientes de software para el sistema a través de Server Update Utility (SUU) y Dell Update Packages (DUP). SUU se entrega en CD por separado para Microsoft Windows y Linux.

    1. Inserte el CD Updates dentro de la unidad de CD de un sistema que ejecuta el sistema operativo, ya sea Windows, Red Hat Enterprise Linux o SUSE LINUX Enterprise Server.
    2. La utilidad Server Update Utility debe iniciar automáticamente. De lo contrario, haga clic en el botón Inicio, luego en Ejecutar, escriba cmd, y presione Entrar. Cuando el símbolo del sistema aparezca, cambie de directorio al raíz de la unidad de CD. Escriba suu -g (para modo gráfico) o suu -h (para mostrar las opciones de ayuda).

    NOTA: En el sistema operativo Red Hat Enterprise Linux o SUSE LINUX Enterprise Server, la opción -g (modo gráfico) puede utilizarse solamente desde el sistema X Window.

    1. Si falla la inicialización de la interfaz GUI, necesitará instalar la versión más reciente del controlador OpenIPMI. Escriba suu -g -installIPMI para instalar la versión más reciente del contolador OpenIPMI.
       
      NOTA: El Instrumentation Service de Server Administrator requiere una versión mínima del controlador del dispositivo OpenIPMI para proporcionar información y funcionalidad basada en IPMI. La versión mínima requerida está definida con base en el sistema operativo. Consulte la Guía del usuario de instalación y seguridad para obtener más información.

    2. Para ver una comparación de repositorio en la GUI, haga clic en el nombre del servidor host mostrado en el árbol a la izquierda. SUU compara lo que se encuentra instalado en el servidor con las versiones de componentes disponibles en el repositorio. Para cada componente en el servidor que está al corriente con la versión en el repositorio de la SUU, ésta última reporta un icono de marca de verificación verde. Para cualquier componente en el servidor que está obsoleto y puede ser actualizado utilizando SUU, se mostrará un icono de advertencia amarillo. Si se encuentran componentes que sean actualizables, se activa el botón Actualizar en la esquina superior derecha de la pantalla.

    3. Haga clic en Actualizar para aplicar todas las actualizaciones que están listadas para el servidor. No puede aplicar las actualizaciones selectivamente. SUU muestra un cuadro de diálogo solicitando confirmación de que desea actualizar los componentes listados. Haga clic en Sí para continuar.

    4. Aparece la ventana de Actualización exitosa, confirmando el éxito de la actualización para SUU y DUP, y es posible que le solicite reiniciar el sistema para completar la actualización.
