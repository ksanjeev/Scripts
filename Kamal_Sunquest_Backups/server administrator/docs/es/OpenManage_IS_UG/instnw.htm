Descripción general

Puede instalar Managed System Software mediante de varios métodos. El CD Dell PowerEdge Installation and Server Management proporciona secuencias de comandos de instalación IPS para instalar, actualizar y desinstalar Dell OpenManage™ Server Administrator y otros componentes de Managed System Software en el sistema administrado. Además, Managed System Software se puede instalar en varios sistemas utilizando una instalación desatendida a lo largo de una red.

NOTA: Consulte el archivo readme_ins.txt en el CD Dell PowerEdge Installation and Server Management para ver una lista de los sistemas admitidos actualmente.Si se detecta una versión anterior del software Systems Management, usted deberá desinstalarla primero. Una vez que se haya completado la desinstalación, tiene que reiniciar el sistema antes de que pueda instalar la versión más reciente del software Systems Management.

Instalación desatendida y silenciosa con secuencia de comandos

El CD Dell PowerEdge Installation and Server Management se puede utilizar para realizar una instalación desatendida y silenciosa con secuencias de comandos de Server Administrator en sistemas que ejecutan sistemas operativos Novell® Netware® admitidos.

Antes de comenzar

  • Lea los requisitos de instalación para asegurarse de que su sistema cumple o supera los requisitos mínimos.

  • Lea la Guía de compatibilidad de Server Administrator. Este documento contiene información de compatibilidad acerca de la instalación y la operación de Server Administrator en diversas plataformas de hardware que ejecutan los sistemas operativos Microsoft® Windows®, NetWare y Red Hat® Enterprise Linux admitidos.

  • Lea el archivo de instalación de Dell OpenManage readme_ins.txt en el CD Dell PowerEdge Installation and Server Management. Este archivo contiene la información más actualizada sobre las nuevas funciones, mejoras y requisitos de hardware, además de la información de los problemas conocidos.

  • Lea el archivo léame de Server Administrator en el CD Dell PowerEdge Installation and Server Management. Este archivo contiene la información más actualizada sobre las versiones del software, firmware y controlador, además de información relacionada con problemas conocidos.

  • Lea las instrucciones para la instalación del sistema operativo.

Requisitos para la instalación

En las secciones siguientes se describen los requisitos generales para instalar Managed System Software.

Versiones de los sistemas operativos admitidos

Managed System Software admite, como mínimo, los siguientes sistemas operativos:

  • Novell NetWare, versión 6.5

NOTA: Consulte el archivo "léame" de Server Administrator en el CD Dell PowerEdge Installation and Sever Management o la Guía de Compatibilidad de Server Administrator de Dell OpenManage en el CD de documentación para obtener la lista detallada más reciente de los servicios de Server Administrator admitidos por cada sistema operativo admitido.

Requisitos del sistema

Managed System Software debe estar instalado en cada uno de los sistemas que se van a administrar. Se pueden administrar entonces todos los sistemas que ejecutan Managed System Software en forma local o remota mediante un explorador de web admitido.

Requisitos del sistema administrado

  • Una de las versiones admitidas de los sistemas operativos.

  • Mínimo de 64 MB de RAM.

  • Mínimo de 256 MB de espacio libre en la unidad de disco duro.

  • Derechos de administrador.

  • Conexión TCP/IP en el sistema supervisado y en el sistema remoto para facilitar la administración de sistemas remotos.

  • El protocolo simple de gestión de red (SNMP) estándar.

  • Mouse, teclado y monitor para administrar sistemas localmente. El monitor requiere una resolución de pantalla mínima de 800 x 600. La resolución recomendada es 1024 x 768.

  • El servicio de acceso remoto de Server Administrator requiere que haya un controlador de acceso remoto (RAC) instalado en el sistema que se va a administrar. Consulte la Guía del usuario de Remote Access Controller 4 de Dell o la Guía del usuario de Dell Embedded Remote Access Controller/MC para conocer todos los requisitos de hardware y de software.

Estándares compatibles de protocolos de administración de sistemas

Debe haber un protocolo estándar de administración de sistemas admitido instalado en el sistema administrado antes de instalar Managed System Software. En los sistemas operativos NetWare admitidos, Managed System Software sólo admite el estándar de administración de sistemas SNMP. El SNMP siempre está instalado. Los estándares WMI y CIM no están disponibles.

NOTA: Para obtener más información sobre la instalación de un estándar de protocolo de administración de sistemas compatible en el sistema administrado, consulte la documentación del sistema operativo.

Procedimientos de instalación

En esta sección se explica cómo instalar, actualizar y desinstalar Managed System Software en un sistema que ejecuta un sistema operativo NetWare compatible.

Instalación y actualización de Managed System Software

En esta sección se explica como instalar y actualizar Managed System Software utilizando las siguientes opciones de instalación:

  • Utilizar el programa de instalación del CD Dell PowerEdge Installation and Server Management para instalar o actualizar el Server Administrator y otro software de Managed System.

  • Use el procedimiento de instalación silenciosa con secuencia de comandos para instalar Server Administrator y otro Managed System Software en múltiples sistemas.

Requisitos previos para instalar o actualizar Managed System Software

Los siguientes son los requisitos previos para instalar Managed System Software:

  • Debe tener privilegios de administrador.

  • El agente SNMP proporcionado con el sistema operativo debe estar instalado en el sistema administrado. Si desea utilizar agentes de asistencia para el agente SNMP, debe instalar la asistencia del sistema operativo para el estándar SNMP antes de instalar el Server Administrator. Para obtener más información sobre la instalación de SNMP, consulte las instrucciones de instalación del sistema operativo que se esté ejecutando en el equipo.

NOTA: La instalación de Managed System Software no se admite en los sistemas que contienen una versión de Systems Management Software anterior a la versión 4.3. Si se descubre una versión anterior del software de Systems Management, debe desinstalar primero esa versión del software de Systems Management.

Instalación y actualización de Managed Systems Software utilizando el CD Dell PowerEdge Installation and Server Management

  1. Inserte el CD Dell PowerEdge Installation and Server Management en la unidad de CD del sistema.

El CD carga automáticamente y se muestran todos los nombres de volumen.

Si no aparece el nombre del volumen, escriba volume en la petición de la consola y oprima <Entrar>.

  1. Registre el nombre del volumen de la unidad de CD.

  2. Escriba load nwconfig y oprima <Entrar>.

  3. En la pantalla Configuración de NetWare, en Opciones de configuración, seleccione Opciones de producto y oprima <Entrar>.

  4. En el menú Otras acciones de instalación, seleccione Instalar un producto no listado y oprima <Entrar>.

NOTA: Es posible que se abra un cuadro de diálogo con una lista de las rutas de acceso seleccionadas previamente tras oprimir <Entrar>. Oprima <Esc> para cerrar el cuadro de diálogo.

Un mensaje predeterminado indicará entonces que el producto se va a instalar desde la unidad A.

  1. Oprima <F3> para especificar una ruta de acceso de instalación diferente.
    Borre A: y escriba nombre_del_volumen:srvadmin\netware, donde nombre_del_volumen es el nombre del volumen asignado a la unidad de CD.

NOTA: No se pueden introducir espacios en el nombre de la ruta de acceso.
  1. Después de introducir la ruta de acceso, oprima <Entrar> para continuar.
    Aparece un mensaje solicitando:
    Indicate which file groups you want installed.

    (Indique los grupos de archivos que desea instalar.)

  2. Seleccione cualquier otro componente que desee instalar. Server Administrator no se incluye en la lista de grupos de archivos porque se instala de manera predeterminada.

NOTA: Los elementos seleccionados para la instalación tienen una X junto a ellos. Si oprime <Entrar> junto a un elemento, éste queda seleccionado. Si oprime la barra espaciadora junto a un elemento, éste cambia alternativamente entre seleccionado y no seleccionado.
  1. Presione <F10> para aceptar los componentes que seleccionó.

  2. A continuación, aparece la pantalla de resumen. Revise la información de resumen y presione <Entrar> para continuar o <Esc> para anular la instalación.
    Se iniciará la instalación del Server Administrator y de los componentes seleccionados.

  3. Se abrirá la pantalla Utilidad de seguridad de Systems Management en la que se preguntará si desea cambiar la configuración de seguridad predeterminada para las operaciones Set de SNMP.

  4. Seleccione una de las siguientes opciones:

    • Oprima <y> para cambiar la contraseña predeterminada del usuario root.
      Introduzca la nueva contraseña y después vuelva a escribir la contraseña para confirmarla.

NOTA: La contraseña predeterminada de root es calvin.
    • Presione <n> para salir de la utilidad de seguridad y restablecer la contraseña de root al valor predeterminado.

Se abre la utilidad de seguridad de Systems Management.

  1. Cuando se le solicite, presione <s> para permitir el establecimiento de operaciones (escritura) SNMP, <n> para rechazar el establecimiento de operaciones (sólo lectura) SNMP, o cualquier tecla para continuar.

El procedimiento de instalación continua, y los mensajes de estado aparecen. Cuando la instalación se complete, se abrirá un cuadro de diálogo informando que la instalación se ha completado.

  1. Oprima <Entrar>.

  2. Oprima <Esc> dos veces y <Entrar> una vez para regresar a la petición de la consola y salir del programa NWConfig.

Realización de una instalación o actualización silenciosa con secuencia de comandos de Managed System Software

Utilice el siguiente procedimiento para instalar silenciosamente Managed System Software por primera vez o para actualizarlo a una versión más reciente.

  1. Inserte el CD Dell PowerEdge Installation and Server Management en la unidad de CD del sistema.

El CD carga automáticamente y se muestra el nombre del volumen.

Si no se muestra el nombre de volumen, escriba volume en la petición de consola presione <Entrar> para mostrar todos los nombres de volumen.

  1. Registre el nombre del volumen de la unidad de CD.

  2. Escriba lo siguiente y oprima <Entrar>:

load nombre_del_volumen:\srvadmin\netware\ominss32.nlm [-noinstallweb] [-noam] [-norac]

La secuencia de comandos de instalación instala todos los archivos necesarios en el sistema administrado.

Puede ver el contrato de licencia (license.txt) en el CD Dell PowerEdge Installation and Server Management en el directorio root.

El valor -noinstallweb es opcional y producirá una instalación que tiene únicamente componentes de línea de comando y no se podrá acceder a ellos por medio de un explorador.

El valor -noam es opcional y producirá una instalación que no tendrá Dell OpenManage Array Manager.

El valor -norac es opcional y producirá una instalación que no tendrá compatibilidad de software de Dell Remote Access Controller.

Desinstalación de Managed System Software

Puede desinstalar Managed System Software del sistema utilizando el menú Opciones del producto de la utilidad NWConfig o mediante la secuencia de comandos de desinstalación silenciosa que se proporciona en el CD Installation and Server Management de Dell PowerEdge.

Desinstalación usando la utilidad NWConfig

  1. En la consola de la línea de comando de NetWare, escriba load nwconfig y oprima <Entrar>.

  2. Seleccione Opciones de producto en el menú Opciones de configuración.

  3. Seleccione Ver/Configurar/Eliminar del menú Otras acciones de instalación .

  4. Seleccione SRVADMIN en el menú Productos instalados actualmente.

  5. Presione <Eliminar> para desinstalar el Server Administrator.

  6. Oprima <Esc> dos veces y <Entrar> una vez para regresar a la petición de la consola y salir del programa NWConfig.

Desinstalación silenciosa

En la consola de la línea de comando de NetWare, escriba omunins.ncf y oprima <Entrar>. La secuencia de comandos de desinstalación desinstala todos los archivos de Managed System Software del sistema administrado.

Note sobre Arranque

Instrumentation Service al inicio

En los sistemas administrados que ejecutan un sistema operativo NetWare, el Instrumentation Service carga varios NLM (módulos cargables de NetWare) durante el arranque de sistema. Es posible que algunos NLM se puedan desinstalar debido a que no se necesitan en un sistema específico. El proceso de desinstalación de los NLM ocurre aproximadamente dos minutos después de que usted inicia o reinicia el sistema. Durante este proceso de desinstalación, aparece un mensaje de descarga de módulo en la consola del sistema NetWare.
