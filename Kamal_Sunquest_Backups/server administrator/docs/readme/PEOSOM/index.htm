Dell PowerEdge™ Software Support Matrix

  Supported Microsoft Windows Operating Systems

  Supported Linux Operating Systems

The Dell PowerEdge Software Support Matrix allows you to identify operating systems and Dell OpenManage™ software that your PowerEdge system supports.

This support matrix offers support information for the following operating systems.

Microsoft® Windows® Operating Systems:

  • Microsoft Windows Server™ 2003 (Web, Standard, Enterprise, x64, and x86 editions)

  • Microsoft Windows Small Business Server 2003 (Standard and Premium editions)

  • Microsoft Windows 2000 (Server and Advanced Server)

  • Microsoft Windows Storage Server x64 (Express, Workgroup, Standard, and Enterprise editions)

  • Microsoft Windows 2000 Professional

  • Microsoft Windows XP Professional

Linux Operating Systems:

  • SUSE® Linux Enterprise Server 9 (SLES 9) Intel® Extended Memory 64 Technology (EM64T) version

  • Red Hat® Enterprise Linux 3 (RHEL 3) AS, ES, and WS editions; Intel EM64T and x86 versions

  • Red Hat® Enterprise Linux 4 (RHEL 4) AS, ES, and WS editions; Intel EM64T and x86 versions

NOTE: For Dell OpenManage information specific to VMware®, see the latest Installation Guide, Release Notes, and the Dell–VMware Compatibility Matrix at www.dell.com/vmware, under the Dell/VMware Resource Center section.
NOTE: In addition to operating system support information, this document also offers information on supported Dell Remote Access Controllers (DRAC) and Dell OpenManage Baseboard Management Controller (BMC).

Supported Microsoft Windows Operating Systems

    • Supported Dell OpenManage Systems Management Consoles

    • Supported Dell Remote Access Controllers

    • Supported Windows Operating Systems for BMC Management Utility (BMU)

Supported Linux Operating Systems
