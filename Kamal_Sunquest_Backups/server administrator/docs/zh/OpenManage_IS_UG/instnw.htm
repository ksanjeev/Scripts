概览

可使用多个方法安装 Managed System Software。Dell PowerEdge Installation and Server Management CD 提供了安装 IPS 脚本来在您的管理型系统上安装、升级和卸载 Dell OpenManage™ Server Administrator 和其它 Managed System Software 组件。此外，您还可以在无人值守的情况下，通过网络在多个系统上安装 Managed System Software。

注：请参阅 Dell PowerEdge Installation and Server Management CD 的 readme_ins.txt 文件了解目前所支持系统的列表。如果检测到以前版本的 Systems Management Software，必须首先卸载它。卸载完成后，必须重新引导系统，然后才能安装最新版本的 Systems Management Software。

无人值守和脚本化无提示安装

在运行支持的 Novell® Netware® 操作系统的系统中，可以使用 Dell PowerEdge Installation and Server Management CD 执行无人值守的脚本化无提示 Server Administrator 安装。

开始之前

  • 阅读安装要求，以确保您的系统可满足或超出最低要求。

  • 阅读《Server Administrator 兼容性指南》。此说明文件包含有关在运行支持的 Microsoft® Windows®、NetWare 和 Red Hat® Enterprise Linux 操作系统的各种硬件平台上安装和运行 Server Administrator 的兼容性信息。

  • 阅读 Dell PowerEdge Installation and Server Management CD 上的 Dell OpenManage 安装 readme_ins.txt 文件。除有关已知问题的信息外，该文件还包含有关新功能、修复和硬件要求的最新信息。

  • 阅读 Dell PowerEdge Installation and Server Management CD 上的 Server Administrator 自述文件。除有关已知问题的信息外，该文件还包含有关软件、固件及驱动程序版本的最新信息。

  • 阅读操作系统的安装说明。

安装要求

以下各节介绍了安装 Managed System Software 的一般要求。

支持的操作系统版本

Managed System Software 最低支持以下操作系统：

  • Novell NetWare 6.5 版

注：请参阅 Dell PowerEdge Installation and Server Management CD 上的 Server Administrator 自述文件或 Documentation CD 上的《Dell OpenManage Server Administrator 兼容性指南》获取各个所支持操作系统上 Server Administrator 服务的最新详细列表。

系统要求

Managed System Software 必须安装在每个要管理的系统上。随后可以通过支持的 Web 浏览器在本地或远程管理运行 Managed System Software 的各个系统。

管理型系统要求

  • 一个支持的操作系统版本。

  • 至少 64 MB RAM。

  • 至少 256 MB 硬盘驱动器可用空间。

  • 管理员权限。

  • 监测的系统和远程系统间的 TCP/IP 连接，以便于远程系统管理。

  • 简单网络管理协议 (SNMP) 标准。

  • 鼠标、键盘和显示器，用于在本地管理系统。显示器要求屏幕分辨率最低为 800 x 600。建议的屏幕分辨率设置为 1024 x 768。

  • Server Administrator Remote Access Service 要求在要管理的系统上安装 Remote Access Controller (RAC)。请参阅《Dell Remote Access Controller 4 用户指南》或《Dell Embedded Remote Access Controller/MC 用户指南》了解软件和硬件的完整要求。

支持的系统管理协议标准

安装 Managed System Software 之前，管理型系统中必须已安装支持的系统管理协议标准。在支持的 NetWare 操作系统中，Managed System Software 只支持 SNMP 系统管理标准。始终安装 SNMP。CIM 和 WMI 标准不可用。

注：有关在管理型系统中安装支持的系统管理协议标准的信息，请参阅操作系统说明文件。

安装步骤

本节说明了如何在运行支持的 NetWare 操作系统的系统上安装、升级和卸载 Managed System Software。

安装和升级 Managed System Software

本节解释如何使用以下安装选项安装和升级 Managed System Software：

  • 使用 Dell PowerEdge Installation and Server Management CD 上的安装程序安装或升级 Server Administrator 和其它 Managed System Software。

  • 使用脚本化安装过程在多个系统中安装 Server Administrator 和其它 Managed System Software。

安装或升级 Managed System Software 的前提条件

以下是安装 Managed System Software 的前提条件：

  • 您必须具有管理员权限。

  • 必须在管理型系统中安装操作系统提供的 SNMP 代理。如果您想使用 SNMP 代理的支持代理，则必须在安装 Server Administrator 之前安装支持 SNMP 标准的操作系统。有关安装 SNMP 的详细信息，请参阅系统中运行的操作系统的安装说明。

注：在包含 4.3 以前版本系统管理软件的系统上不支持安装 Managed System Software。如果检测到以前版本的系统管理软件，必须首先卸载系统上的系统管理软件。

使用 Dell PowerEdge Installation and Server Management CD 安装和升级 Managed Systems Software

  1. 将 Dell PowerEdge Installation and Server Management CD 插入系统的 CD 驱动器中。

CD 会自动装载并且会显示所有的卷名。

如果没有显示卷名，在控制台提示符处键入 volume 并按 <Enter>。

  1. 记录 CD 驱动器的卷名。

  2. 键入 load nwconfig 并按 <Enter> 键。

  3. 在“NetWare Configuration”（NetWare 配置）屏幕中，“Configuration Options”（配置选项）下，选择“Product Options”（产品选项）并按 <Enter>。

  4. 在“Other Installation Actions”（其它安装操作）菜单中，选择“Install a product not listed”（安装未列出产品）并按 <Enter> 键。

注：按 <Enter> 键后系统可能打开对话框，列出先前选定的路径。按 <Esc> 键关闭此对话框。

默认信息随后会指出将从驱动器 A 安装产品。

  1. 按 <F3> 键以指定不同的安装路径。
    删除 A: 并键入卷名:srvadmin\netware，其中卷名是分配给 CD 驱动器的卷名。

注：您不能在路径名称中输入空格。
  1. 输入路径后，按 <Enter> 键以继续。
    将显示一条信息，说明：
    Indicate which file groups you want installed.（指出要安装的文件组。）

  2. 选择您要安装的任何其它组件。文件组列表中未包含 Server Administrator，因为它将被默认安装。

注：已选定要安装的项目旁边有 X。按项目旁边的 <Enter> 键可选择该项目。按项目旁边的空格建可在已选定和未选定选项之间切换。
  1. 按 <F10> 接受所选组件。

  2. 接下来将会显示摘要屏幕。查看摘要信息并按 <Enter> 继续，按 <Esc> 异常中断安装。
    开始安装 Server Administrator 和选定的组件。

  3. “Systems Management Security Utility”（Systems Management 安全性公用程序）屏幕将会打开，并询问您是否要更改 SNMP Set 操作的默认安全性设置。

  4. 选择以下选项之一：

    • 按 <y> 键更改 root 的默认密码。
      键入新密码，并再次键入该密码以确认。

注：root 的默认密码是 calvin。
    • 按 <n> 离开安全性公用程序并将 root 的密码重设为默认值。

Systems Management 安全性公用程序会打开。

  1. 提示时，按 <y> 允许 SNMP 设置（写）操作，<n> 不允许 SNMP 设置操作（只读）或按任何键继续。

安装过程会继续，并显示状态信息。安装完成后，将会打开一个对话框，表明安装已完成。

  1. 按 <Enter> 键。

  2. 按 <Esc> 键两次并按 <Enter> 键一次以返回至控制台提示符并退出 NWConfig 程序。

执行 Managed System Software 的脚本化无提示安装或升级

首次无提示安装 Managed System Software 或升级至 Managed System Software 的更新版本时，请使用以下步骤。

  1. 将 Dell PowerEdge Installation and Server Management CD 插入系统的 CD 驱动器中。

CD 会自动装载并且会显示卷名。

如果没有显示卷名，在控制台提示符处键入 volume 并按 <Enter> 显示所有卷名。

  1. 记录 CD 驱动器的卷名。

  2. 键入以下命令并按 <Enter>：

load 卷名:\srvadmin\netware\ominss32.nlm [-noinstallweb] [-noam] [-norac]

安装脚本将所有必要的文件安装至管理型系统。

可以查看许可协议 (license.txt)（位于 Dell PowerEdge Installation and Server Management CD 的根目录中）。

Cnoinstallweb 设置是可选的，会导致只有命令行组件的安装并且不能通过浏览器访问。

Cnoam 设置是可选的，会导致没有 Dell OpenManage Array Manager 的安装。

Cnorac设置是可选的，会导致没有 Dell Remote Access Controller 软件支持的安装。

卸载 Managed System Software

通过使用 NWConfig 公用程序中的“Product Options”（产品选项）菜单或使用 Dell PowerEdge Installation and Server Management CD 上提供的无提示卸载脚本，您可以从系统卸载 Managed System Software。

使用 NWConfig 公用程序卸载

  1. 在 NetWare 命令行控制台上键入 load nwconfig 并按 <Enter> 键。

  2. 从“Configuration Options”（配置选项）菜单中选择“Product Options”（产品选项）。

  3. 从“Other Installation Actions”（其它安装操作）菜单中选择“View/Configure/Remove”（查看/配置/删除）。

  4. 从“Currently Installed Products”（当前安装的产品）菜单选择 SRVADMIN。

  5. 按 <Delete> 卸载 Server Administrator。

  6. 按 <Esc> 键两次并按 <Enter> 键一次以返回至控制台提示符并退出 NWConfig 程序。

执行无提示卸载

在 NetWare 命令行控制台，键入 omunins.ncf 并按 <Enter> 键。卸载脚本将从管理型系统上卸载所有 Managed System Software 文件。

启动时的注释

启动时的 Instrumentation Service

在运行 NetWare 操作系统的管理型系统上，Instrumentation Service 会在系统启动时载入多个 NetWare 可加载模块 (NLM)。有些 NLM 可能会自动卸载，因为在特定系统上不需要它们。卸载 NLM 的过程在引导或重新启动系统后的大约两分钟时开始。在此卸载过程中，模块卸下信息会显示到 NetWare 系统控制台。
