Demo Download and Installation Instructions

Attention Netscape Users:

Due to several documented FTP issues involving Netscape, we strongly recommend using Internet Explorer to download the demo files.  If you do not have access to Internet Explorer, the following information may be of use.  When you click on the download link, Netscape's default response is to try to open the file instead of downloading it, so you may see ASCII characters on your screen. 

Depending on which version of Netscape you are using, one of the following may be helpful.

  • Holding down the shift key while clicking on the hyperlink tells some versions of Netscape to download the file rather than  listing it as ASCII text on the screen.
  • Right-clicking on the link will bring up a shortcut menu that allows you to save the file to disk.

If neither of these techniques is successful, please try using Internet Explorer to download the file.

While you are downloading, minimizing the download dialog box to your taskbar will allow you to continue working on other tasks.  Once the download is complete:

SLIM-Estimate, SLIM-Metrics, and SLIM-MasterPlan guided tours:

1. Simply click on the downloaded file to install the guided tour file to your PC.  Shortcuts will be placed on your desktop and in the programs folder.  To run the demo, first make sure you are connected to the Internet and that you have a high-speed connection (DSL or T1 line).  When you run the demo, the sound files will be streamed from the web.

2. To run the demo, you need to have Windows Media Player installed on your PC.  A download link is provided on the introductory screen of the demo.
